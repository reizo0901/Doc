6	共有フォルダの情報をSambaに設定する	Tera Term	"1. 下記コマンドを実行し、ファイルの内容を確認する
　cat /etc/samba/smb.conf

　vi /etc/samba/smb.conf
2. キーボードの「i」を押し、編集可能状態にする
　カーソルポイントを編集したい場所に移動させた後、設定ファイルの中身を以下のように編集する(書き加える)
　[DATA2]　　　　　　　　　　
   comment = 　　　
   path = /DATA2　　　　　
   guest ok = yes
   browseable = yes　
　 read only = no 
3. 編集完了後、Escキー→:wq→Enterにて保存する"	"下記コマンドを実行し、「Loaded services file OK.」と表示されることを確認
testparm"					"[ディレクトリ名] ・・・共有名
comment ・・・コメント（無くて良い）
path ・・・共有するディレクトリのpath
guest ok ・・・ゲストユーザーの許可
browseable ・・・ブラウザーへの表示を許可
read only ・・・ディレクトリ、ファイルへの書き込みを許可

ファイルを保存せずに終了する場合(閲覧だけする場合)→:q

ファイルを保存して終了する場合→:wq"
